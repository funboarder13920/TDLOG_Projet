- Le jeu se lance en tapant « python kahmate.py » dans la console
- Le joueur bleu place en premier ses joueurs en cliquant sur l’un d’entre eux puis sur une case dans la zone (cf. règles) puis clique sur le pouce bleu lorsqu’il a fini de placer ses joueurs
- C’est ensuite au tour du joueur rouge
- Ensuite, l’interface demande à l’équipe tirée au sort de commencer à joueur. Il faut d’abord sélectionner un joueur. Lorsqu’un joueur est sélectionné ses informations s’affichent au-dessus du plateau. Il faut ensuite cliquer sur une case de la grille pour effectuer une action qui sera automatiquement déduite. 
- Lorsque le joueur a fini de jouer il clique sur le pouce pour donner la main au joueur suivant.

Particularités : 
- il est impossible de faire un tir en avant sur la case directement devant soi.
- Le tirage des cartes pour résoudre les actions se fait aléatoirement.
